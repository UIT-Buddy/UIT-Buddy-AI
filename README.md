### Features

*   **FastAPI Server**: High-performance asynchronous API server.
*   **Chatbot Integration**: Handles intelligent, context-aware student queries.
*   **Academic RAG**: Utilizes LightRAG and Neo4j for semantic search and retrieval of university information, schedules, and career roadmaps.
*   **Backend Proxy**: Securely routes authenticated requests to the main Java Spring Boot backend.

### Technology Stack

*   **Framework**: FastAPI, Uvicorn
*   **AI/RAG**: LightRAG, Tiktoken
*   **Graph Database**: Neo4j
*   **HTTP Client**: HTTPX
*   **Validation**: Pydantic
